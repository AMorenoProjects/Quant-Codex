1. Solución Cerrada (Closed Form)

[!abstract]

Define la base matemática estricta y el álgebra lineal subyacente para entrenar modelos predictivos mediante matrices en RegresionLineal, permitiendo encontrar los coeficientes óptimos de forma analítica.

1. Solución Cerrada (Closed Form)¶

La solución cerrada permite calcular los pesos o coeficientes óptimos (β^\hat{\beta}β^​) de un modelo predictivo directamente, resolviendo el sistema matricial sin necesidad de emplear algoritmos iterativos como el DescensoDeGradiente.

[!math-blue] Ecuación Óptima de Coeficientes

β^=(XTX)−1XTy\hat{\beta} = (X^T X)^{-1} X^T yβ^​=(XTX)−1XTy
(1)

  • XXX: Matriz de características (las filas son datos históricos, las columnas son variables predictoras).

  • yyy: Vector con los valores objetivo reales.

  • XTX^TXT: Matriz transpuesta de XXX.

  • (XTX)−1(X^T X)^{-1}(XTX)−1: Inversa de la matriz cuadrada resultante del producto.

2. Derivación Matemática¶

El objetivo fundamental es minimizar el error, definido estadísticamente como la Suma de Cuadrados de los Residuos (RSSRSSRSS).

[!math-green] Suma de Cuadrados de los Residuos (RSS)

RSS=(y−Xβ)T(y−Xβ)RSS = (y - X\beta)^T (y - X\beta)RSS=(y−Xβ)T(y−Xβ)
(2)

Para encontrar el punto mínimo de la curva de error, se aplica cálculo diferencial: se deriva respecto al vector β\betaβ y se iguala a cero. Esto produce las denominadas Ecuaciones Normales:

[!math-red] Derivada y Ecuaciones Normales

∂RSS∂β=−2XT(y−Xβ)=0\frac{\partial RSS}{\partial \beta} = -2X^T (y - X\beta) = 0∂β∂RSS​=−2XT(y−Xβ)=0
(3)

Al despejar β\betaβ de este sistema matricial, se obtiene exactamente la fórmula de la solución cerrada descrita en el punto anterior.

3. Matriz Sombrero (Hat Matrix)¶

Una vez entrenado el modelo con β^\hat{\beta}β^​, se generan las predicciones y^\hat{y}y^​ multiplicando los datos de entrada XXX por los pesos optimizados:

[!math] Ecuación Base de Predicción

y^=Xβ^\hat{y} = X \hat{\beta}y^​=Xβ^​
(4)

Si se sustituye β^\hat{\beta}β^​ por su desarrollo analítico completo, se revela el operador de proyección:

[!math-purple] Hat Matrix (HHH)

y^=[X(XTX)−1XT]y\hat{y} = \left[ X(X^T X)^{-1} X^T \right] yy^​=[X(XTX)−1XT]y
(5)

[!info] Proyección Ortogonal

El bloque matricial entre corchetes es la matriz HHH (Hat Matrix). Geométricamente, HHH actúa como un operador de proyección ortogonal que “aplasta” el vector multidimensional yyy original para que encaje como una sombra exacta sobre el plano o subespacio generado por las variables XXX.

4. Colapso por Colinealidad¶

Al programar o implementar estos modelos a nivel código, existe un punto crítico de fallo directamente relacionado con las propiedades de las matrices.

[!danger] Matriz Singular y Pérdida de Rango

Para calcular la solución cerrada, la matriz XXX debe tener rango completo (full rank). Esto requiere independencia lineal absoluta entre todas las columnas (variables).

Si se introducen dos variables perfectamente correlacionadas (colinealidad perfecta, ej. precio en dólares y su conversión exacta a euros), ocurre un fallo algebraico:

  1. Redundancia Matemática: Una columna se vuelve dependiente de otra.

  2. Determinante Cero: El determinante de la matriz (XTX)(X^T X)(XTX) se vuelve 0.

  3. Matriz Singular: Resulta matemáticamente imposible calcular su inversa (XTX)−1(X^T X)^{-1}(XTX)−1, equivalente a un intento de división por cero.

  4. Fallo de Ejecución: El algoritmo lanza un error, ya que no existe un criterio matemático para repartir el peso β^\hat{\beta}β^​ entre dos variables que aportan idéntica información.
